---
title: Redirect Netcup Domain to Cloudflare
slug: /redirectNetcupDomainToCloudflare
---

Nameserver löschen:
```
root-dns.netcup.net
second-dns.netcup.net
third-dns.netcup.net
```
Nameserver hinzufügen:
```
alan.ns.cloudflare.com
luciane.ns.cloudflare.com
```
